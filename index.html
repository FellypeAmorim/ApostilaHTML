Imagens

Podemos adicionar imagens utilizando o comando 'img src""'.
ex: img src"imagem/ferrari.jpg"
A imagem deve estar dentro da pasta do projeto.

Também podemos utilizar uma imagem direto da internet utilizando um link utilizando a mesma estrutura.

Podemos ajustar a largura com o "width" e a altura com o "height".
ex. img src="imagens/htmlimg.jfif" width:"300" height:"400""

Tabelas

Para iniciarmos uma tabela utilizamos o comando "table", podemos incrementar adicionando bordas com o comando "border" e precisamos adicionar mais duas estruturas dentro de "table", as linhas com "tr"(table roll) e os dados com "td"(table data). Podemos definir os titulos de nossa tabela com "th"(table header).

Formulários

Para criarmos um formulário utilizamos o comando "form" e dentro de formas colocamos os itens deste formulario utilizando o comando "imput" seguido dos atributos "type" e "name". Se utilizarmos o "type" password, os caracteres serão substituidos por asteristicos.
Podemos criar um botão para nosso formulário utilizando o atributo button ou submit porém não precisamos definir um "name", ao ivés disso utilizamos o atributo "value" para definir o texto no botão. ex. imput type:"text" name:"login"
